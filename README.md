This is my attempt at a Java implementation of a calculator that can perform operations on real-valued matrices. I may attempt to extend this implementation to operate on complex-valued matrices if I become motivated enough to do it.

List of operations to implement:
- Matrix Addition
- Matrix Multiplication
- Matrix Transpose
- Dot Products on Vectors
- Gaussian Elimination
- Determinants
- Matrix Inverse
- Null Space Basis Computations
- Eigenvalue/Eigenvector Computations
- (Orthogonal) Diagonalization
- Singular Value Decomposition
